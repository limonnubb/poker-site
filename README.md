# 🃏 Poker Online

Современный онлайн-покер с тёмной темой и плавными анимациями.

## 🚀 Запуск

```bash
cd poker-site
npm install
npm run dev
```

Открой http://localhost:3000 в браузере.

## 🎮 Функционал

- **Главная страница**: Кнопка "Играть", ввод никнейма, настройки
- **Покерный стол**: Карты с анимацией, отображение игроков, ставки
- **Действия**: Fold, Call, Raise с анимациями
- **Настройки**: Звук, камера (WebRTC), выбор количества игроков (2-8)
- **Логика**: Раздача карт, ход игроков, определение победителя

## 🛠 Технологии

- Next.js 14
- React 18
- Tailwind CSS
- Framer Motion
- Zustand (state management)
- WebRTC (веб-камера)

## 📁 Структура

```
poker-site/
├── src/
│   ├── app/          # Next.js страницы
│   ├── components/   # UI компоненты
│   ├── hooks/        # Хуки (камера, игра)
│   ├── store/        # Zustand store
│   └── utils/        # Утилиты (звуки)
├── public/           # Статика
└── package.json
```

## ⚙️ Настройки

- **Sound**: Вкл/выкл звуковые эффекты
- **Camera**: Подключение веб-камеры
- **Players**: Количество игроков за столом (2-8)
